# CWL+MPI thesis documents

This folder contains documents related to the CWL+MPI thesis.

The `cwl-mpi-thesis-bibliography.bib` file contains the bibliography
generated with Zotero 9.0.4 on Linux. Zotero was configured to save
the bibliography in BibTeX format, on disk in this folder. Then, the
bibliography file was uploaded to Overleaf (repeated…), due to lack
of a Pro license.

The bibliography file uses the IEEE bibliography style. It uses DOI
as the identifier of each entry (e.g., `doi:10.1109/ICSC.2016.7720420`).
