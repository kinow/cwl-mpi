# CWL & MPI — Images

This folder contains the images used in the thesis.

The file `cwl_mpi.svg` was created with Inkscape 1.4.3 on Linux.
It contains the vector images. Each of these images was used to
create a raster PNG image, in 300 dpi resolution.

The exported images are found in this folder and were used in
the LaTeX document.

Only the CWL logo was imported into the document. Everything else
was created by hand, inside Inkscape.

The gear used to represent executables was created using this 2008
Inkscape tutorial by Nicu B., probably made for Inkscape 0.46/0.47:
“Drawing gears in Inkscape” https://howto.nicubunu.ro/gears/
